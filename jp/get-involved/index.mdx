## コミュニティチャンネル

公式チャンネルを通じてQuaiネットワークコミュニティとつながります：

| チャンネル | 目的 | リンク |
|---------|---------|------|
| Discord | メインコミュニティハブ | [Discordに参加](https://discord.gg/quai) |
| Twitter | ニュースとアップデート | [@QuaiNetworkをフォロー](https://twitter.com/QuaiNetwork) |
| Telegram | グローバルコミュニティチャット | [Telegramに参加](https://t.me/QuaiNetwork) |
| YouTube | 教育コンテンツ | [チャンネル登録](https://youtube.com/@QuaiNetwork) |
| GitHub | オープンソースコード | [リポジトリを見る](https://github.com/dominant-strategies) |

## コミュニティガバナンス

Quaiネットワークは、コミュニティが所有し運営するシステムの構築に取り組んでいます。コミュニティメンバーがネットワークの将来の開発に発言権を持てるようにするガバナンスメカニズムに取り組んでいます。

ガバナンス提案と投票メカニズムに関する更新情報にご注目ください。


## Quaiで構築

Quaiネットワーク上でアプリケーションを構築することに興味のある開発者は、包括的な[開発者ドキュメント](/jp/build)を参照して開始できます。
