# Quai ネットワーク

Quai ネットワークは、新しいグローバル通貨システムとして機能するように設計された、スケーラブルでプログラム可能なプルーフ・オブ・ワーク・ブロックチェーンです。通貨とエネルギーを統合することで、Quai は世界初の分散型エネルギー・ドル Qi を提供します。Qi は安定性があり、スケーラブルで、実世界の商業のために構築されています。Quai は、エントロピー最小値証明と呼ばれる次世代のプルーフ・オブ・ワーク・コンセンサスメカニズムを活用して、ブロック競合を排除し、稲妻のような最終性を提供します。

## エネルギー・ベースの「ステーブルコイン」

Quai では、プルーフ・オブ・ワークは攻撃者からネットワークを保護するためだけでなく、[現実世界の需要とエネルギーコストのオラクル](/ja/learn/tokenomics/token-dynamics/token-dynamics)としても機能し、完全に[エネルギーに連動した分散型「ステーブルコイン」](/ja/learn/tokenomics/qi-emissions)の作成を可能にします。このトークンはどの資産にもペッグまたはバックされていませんが、代わりに[市場インセンティブ](/ja/learn/tokenomics/token-dynamics/token-dynamics)を独占的に使用して、トークンの価格を一貫してエネルギーコストに向けて推進します。

エネルギーは人間の活動と人工知能の両方への最大の単一投入です。人間と機械の両方にとってのエネルギーの内在的価値は、新しい通貨を構築するための理想的な基盤となっています。

Quai ネットワークは、[Hacash](https://hacash.money/) や [Themelio](https://docs.melproject.org/concepts/melmint) などのプロトコルと共に、分散型プルーフ・オブ・ワーク「ステーブルコイン」の新興カテゴリのリーダーとなっています。

## 解放されたスケーラビリティ

[エントロピー最小値証明 (PoEM)](/ja/learn/advanced-introduction/poem/poem) と呼ばれる新しいプルーフ・オブ・ワーク・フォーク選択ルールを使用することで、Quai ネットワークはコンセンサスを維持しながら、グローバル商業をすべてサポートするのに十分な[実行シャード](/ja/learn/advanced-introduction/poem/sharding/execution-sharding)を調整することができます。この特性により、Quai ネットワークは需要に応じてネットワークにシャードを動的に追加し、絶えず成長する需要を満たすことができます。Quai にシャードを追加すること（したがってスループットを増加させること）の唯一のトレードオフは、[クロスシャード決済時間の増加](/ja/learn/advanced-introduction/poem/sharding/dynamic-sharding#time-to-global-settlement)です。

ネットワーク内のすべてのシャード間のトラストレス・ブリッジは、[マイナーを使用してシャード間の客観的なリンクを作成する](/ja/learn/advanced-introduction/merged-mining/coincident-blocks)ことによって実現されます。

Quai ネットワークのネイティブで事前定義されたスケーリングは、需要の増加に伴って輻輳を緩和するメカニズムを作成し、使用量の増加があっても料金を永続的に低く保ちます。
