### ウォレットにトークンを追加

以下のボタンを使用して、BaseまたはEthereum上のウォレットにQUAIトークン（Symbiosis表現）を追加します。ウォレット（例：MetaMask）がインストールされていることを確認してください。ボタンをクリックすると、正しいネットワークに切り替わり、「トークンを追加」ダイアログが表示されます。

<div style={{ display: 'flex', gap: '12px', flexWrap: 'wrap' }}>
  <button
    onClick={async () => {
      const tokenLogo = 'https://pbs.twimg.com/profile_images/1882168961061531648/fsmniyg7_400x400.jpg'
      const pickEvmProvider = () => {
        if (typeof window === 'undefined' || !window?.ethereum) return null
        const candidates = Array.isArray(window.ethereum.providers) && window.ethereum.providers.length
          ? window.ethereum.providers
          : [window.ethereum]

        const prioritized = candidates.find((provider) => provider?.isMetaMask || provider?.isCoinbaseWallet || provider?.isBraveWallet)
        return prioritized ?? candidates[0]
      }

      const provider = pickEvmProvider()

      if (!provider || typeof provider.request !== 'function') {
        alert('EVMウォレットが検出されませんでした。MetaMask、Coinbase Wallet、または他のEVM互換ウォレットをインストールしてください。')
        return
      }

      if (provider.__quaiAddTokenPending) {
        alert('既に開いているウォレットポップアップを先に完了してから、もう一度お試しください。')
        return
      }

      try {
        provider.__quaiAddTokenPending = true
        await provider.request({ method: 'eth_requestAccounts' })

        try {
          await provider.request({
            method: 'wallet_switchEthereumChain',
            params: [{ chainId: '0x2105' }],
          })
        } catch (switchErr) {
          if (switchErr?.code === 4902) {
            await provider.request({
              method: 'wallet_addEthereumChain',
              params: [{
                chainId: '0x2105',
                chainName: 'Base',
                nativeCurrency: { name: 'Ether', symbol: 'ETH', decimals: 18 },
                rpcUrls: ['https://mainnet.base.org'],
                blockExplorerUrls: ['https://basescan.org'],
              }],
            })
            await provider.request({
              method: 'wallet_switchEthereumChain',
              params: [{ chainId: '0x2105' }],
            })
          } else if (switchErr?.code === -32601) {
            throw new Error('接続されたウォレットはネットワークを切り替えることができません。MetaMaskまたは他のEVMウォレットをお試しいただくか、手動でトークンを追加してください。')
          } else if (switchErr?.code === 4001) {
            throw new Error('ウォレットでネットワーク切り替えが拒否されました。')
          } else {
            throw switchErr
          }
        }

        await provider.request({
          method: 'wallet_watchAsset',
          params: {
            type: 'ERC20',
            options: {
              address: '0x5c97D726bf5130AE15408cE32bc764e458320D2f',
              symbol: 'QUAI',
              decimals: 18,
              image: tokenLogo,
            },
          },
        })

        alert('QUAIがウォレットに追加されました。')
      } catch (e) {
        console.error(e)
        alert(e?.message || 'トークンの追加に失敗しました。接続後に再試行するか、ウォレットで手動で追加してください。')
      } finally {
        provider.__quaiAddTokenPending = false
      }
    }}
    style={{ padding: '10px 14px', borderRadius: 8, border: '1px solid #ddd', background: 'var(--secondary-bg)', color: 'var(--primary-text)', cursor: 'pointer' }}
  >
    Base上のQUAIをBaseウォレットに追加
  </button>

## Symbiosis Bridge

Symbiosisは、複数のブロックチェーンにわたって流動性をプールするクロスチェーンAMM DEXで、ユーザーが単一のトランザクションでサポートされている任意のチェーン上の任意のトークンを交換できるようにします。

<Card title="Symbiosis App" icon="bridge" href="https://symbiosis.finance">
  Symbiosisブリッジインターフェースにアクセスして、Quai Networkへ資産を転送したり、Quai Networkから資産を転送したりできます。
</Card>

## 統合に関する注意事項

- **CoinGecko Platform ID:** `quai-network`
- **Gas Token ID:** `quai-network`
- Quaiはクロスチェーン操作にCyprus1ゾーンを使用します
- Quai上のトランザクションは、アカウントベース（Quai）トークンに対して標準的なEVM互換署名を使用します

<Note>
  USDTは現在Quai Network上のネイティブトークンとして利用できません。他のチェーンからUSDTをブリッジすると、Symbiosis Synthesisコントラクトを通じて合成資産として表されます。
</Note>
