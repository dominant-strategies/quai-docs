When using the [**Interface.parseError**](/content/classes/Interface.mdx#parseerror) to automatically match an
 error for a call result for parsing, an **ErrorDescription** is returned.

## Properties

| Property | Modifier | Type | Description |
| :------ | :------ | :------ | :------ |
| `args` | `readonly` | [`Result`](/content/classes/Result.mdx) | The arguments passed to the Error with `revert`. |
| `fragment` | `readonly` | [`ErrorFragment`](/content/classes/ErrorFragment.mdx) | The matching fragment. |
| `name` | `readonly` | `string` | The name of the Error. |
| `selector` | `readonly` | `string` | The selector for the Error. |
| `signature` | `readonly` | `string` | The full Error signature. |
