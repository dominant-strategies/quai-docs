```ts
function parseUnits(value, unit?): bigint
```

Converts the //decimal string// `value` to a BigInt, assuming `unit` decimal places. The `unit` may the number of
decimal places or the name of a unit (e.g. `"gwei"` for 9 decimal places).

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `value` | `string` | The value to convert. |
| `unit`? | `string` \| [`Numeric`](/content/type-aliases/Numeric) | The unit to convert from. Default is `18` |

## Returns

`bigint`

The converted value.

## Throws

If the unit is invalid.

## Throws

If the value is not a string.
