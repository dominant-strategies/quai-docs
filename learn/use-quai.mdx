Quai Network started in 2018 with the design of a scalable proof-of-work architecture and a plan for a hierarchical, merge-mined chain to scale PoW.
It emerged from the [Dr. K](https://x.com/mechanikalk)’s work and GridPlus and was a research project at the University of Texas at Austin, where NSF-supported research in 2019 helped advance the concept into a development comapny called Dominant Strategies.
Since then, it has been developed and improved with innovations like Proof-of-Entropy-Minima [(PoEM)](https://arxiv.org/abs/2303.04305), a dual-token model (Qi and Quai), multi-algorithm mining, and workshare considerations. Quai Network mainnet began with the genesis block on February 4, 2025.

## Mine Quai

Quai Network uses the KawPow mining algorithm and is merge-mined with SHA-256 and Scrypt. Miners can select to mine either QUAI or QI.

### Solo Mining (StratumX)

Connect to decentralized mining infrastructure with no fees, no KYC, with full self-custody and mining customization including Qi mining and [Locked Mining Rewards](https://docs.qu.ai/learn/advanced-introduction/liquid-mining-tokens).

| Resource | Type | Features |
| :--- | :--- | :--- |
| [StratumX.org](https://www.stratumx.org/) | Platform | Connect SHA-256, Scrypt, and KawPow rigs directly (no node required) |
| [Revenue Calculator](https://www.stratumx.org/calculator) | Tool | Estimate profitability for different algorithms |
| [Setup Guide](https://docs.qu.ai/guides/client/solo-mining) | Documentation | Run a StratumX node for QUAI mining with KawPow, Scrypt, SHA-256|
