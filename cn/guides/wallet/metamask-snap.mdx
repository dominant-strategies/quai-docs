## 概述

Quai 钱包 snap 允许 MetaMask 用户在 Quai 主网上发送和接收 QUAI 代币。它还使 Quai dApp 能够与已安装该 snap 的 MetaMask 用户进行交互。这种集成为 MetaMask 用户提供了一种无需单独钱包即可访问 Quai Network 生态系统的无缝方式。

## 安装

### 步骤 1：从 MetaMask Snaps 目录安装

1. 访问 [MetaMask Snaps 目录](https://snaps.metamask.io)
2. 搜索 "Quai Wallet" 或导航到 Quai 钱包 snap
3. 点击 "添加到 MetaMask" 安装 snap
4. 按照 MetaMask 提示批准安装

### 步骤 2：访问您的 Quai 钱包

安装后，您可以通过 MetaMask 访问您的 Quai 钱包：

1. 打开 MetaMask 扩展程序
2. 点击 MetaMask 窗口右上角的三个虚线（⋮⋮⋮）
3. 在下拉列表中点击 "Snaps"
4. 在列表中点击 "Quai Wallet"
5. 等待 snap 生成您的 Quai 钱包（这需要几秒钟）

<Warning>
如果您在 Snaps 列表中没有看到 "Quai Wallet"，请确保您已从 Snaps 目录成功安装了 snap。
</Warning>

## 钱包生成

Quai 钱包 snap 从您的 MetaMask 密钥确定性地生成您的钱包地址。这意味着：

- **无需助记词**：您的 Quai 地址源自您现有的 MetaMask 账户
- **一致的地址**：只要您拥有 MetaMask 密钥，您将始终生成相同的 Quai 地址
- **安全**：您的 Quai 钱包继承了 MetaMask 设置的安全性

## 使用方法

### 发送 QUAI 代币

1. 通过 MetaMask Snaps 界面访问您的 Quai 钱包
2. 点击 "发送" 或 "转账"
3. 输入接收方的 Quai 地址
4. 指定要发送的 QUAI 数量
5. 查看交易详情
6. 通过 MetaMask 确认交易

### 接收 QUAI 代币

1. 通过 MetaMask Snaps 界面访问您的 Quai 钱包
2. 从钱包界面复制您的 Quai 地址
3. 与他人分享此地址以接收 QUAI 代币
4. 收到的交易将出现在您的交易历史中

## 安全注意事项

- 您的 Quai 钱包与您的 MetaMask 钱包一样安全
- 在确认之前始终验证交易详情
- 保护好您的 MetaMask 助记词 - 它控制着您的以太坊和 Quai 钱包
- 小心钓鱼攻击，只从官方来源安装 snap

## 故障排除

### Snap 未出现
- 确保您已从 MetaMask Snaps 目录成功安装了 snap
- 尝试刷新 MetaMask 或重新启动浏览器
- 检查您是否使用支持的 MetaMask 版本

### 钱包生成问题
- 等待几秒钟让钱包生成
- 如果问题持续存在，尝试删除并重新安装 snap
- 确保您的 MetaMask 已正确设置并解锁

### 交易问题
- 验证您有足够的 QUAI 余额进行交易
- 确保接收方地址是有效的 Quai 地址

## 常见问题解答

<AccordionGroup>
  <Accordion title="什么是 MetaMask Snap？">
    MetaMask Snaps 是扩展 MetaMask 功能的程序。它们允许开发者向 MetaMask 钱包添加新功能和能力，例如支持像 Quai Network 这样的新区块链。
  </Accordion>
  
  <Accordion title="我需要为 Quai Network 准备单独的钱包吗？">
    不一定。Quai 钱包 snap 允许您直接通过现有的 MetaMask 钱包与 Quai Network 进行交互。您无需安装或管理单独的钱包应用程序。但是，对于希望获得最佳 dApp 体验的用户，我们建议安装 [Pelagus 钱包](https://pelaguswallet.io)。
  </Accordion>
  
  <Accordion title="使用 Quai 钱包 snap 安全吗？">
    是的，Quai 钱包 snap 已经过 Hacken 的审计，并在设计时考虑了安全性。它从您现有的 MetaMask 密钥确定性地生成您的 Quai 地址，因此它继承了您 MetaMask 设置的安全性。
  </Accordion>

  <Accordion title="我可以在 Quai 钱包 snap 中存储其他代币吗？">
    目前还不行。您可以在您的地址上接收 QRC20 代币，但目前无法发送它们。这将在下次更新中修复。
  </Accordion>

  <Accordion title="我可以使用 Quai 钱包 snap 与 Quai dApp 交互吗？">
    目前还不行。Quai dApp 必须专门为 Quai 钱包 snap 添加支持。这与直接集成 MetaMask 的过程不同。当 dApp 添加了对 Quai 钱包 snap 的支持时，我们会在此更新。如果您有兴趣与 Quai dApp 交互，我们建议您安装 [Pelagus 钱包](https://pelaguswallet.io)。
  </Accordion>

  <Accordion title="我可以在 Quai 钱包 snap 中创建新地址吗？">
    不可以，Quai 钱包 snap 的设计考虑了基本功能。您的 Quai 地址从您的 MetaMask 密钥生成，目前每个 MetaMask 助记词只生成一个地址。如果您希望生成多个 Quai 地址，我们建议您安装 [Pelagus 钱包](https://pelaguswallet.io)。
  </Accordion>

  <Accordion title="我在 Snaps 目录中找不到 Quai 钱包 snap。我该怎么办？">
    确保您访问的是官方 MetaMask Snaps 目录 https://snaps.metamask.io。如果仍然找不到，snap 可能暂时不可用，或者您可能需要更新 MetaMask 版本。
  </Accordion>
  
  <Accordion title="为什么生成我的 Quai 钱包需要几秒钟？">
    snap 需要使用加密函数从您的 MetaMask 密钥派生您的 Quai 地址。此过程需要几秒钟，但每个会话只发生一次。
  </Accordion>
  
  <Accordion title="使用相同的 MetaMask 账户安装 snap 时，我每次都会获得相同的 Quai 地址吗？">
    是的！由于您的 Quai 地址是从您的 MetaMask 密钥确定性生成的，只要您使用相同的 MetaMask 账户，您将始终获得相同的地址。
  </Accordion>
  
  <Accordion title="我可以将 QUAI 代币发送到任何以太坊地址吗？">
    不可以，您只能将 QUAI 代币发送到有效的 Quai 地址。Quai 地址看起来像以太坊地址，但以 0x00 开头。
  </Accordion>
  
  <Accordion title="我如何知道交易是否成功？">
    您可以在 MetaMask 内的 Quai 钱包界面中检查交易状态，或在 [Quaiscan](https://quaiscan.io) 上查看。
  </Accordion>
  
  <Accordion title="如果我将 QUAI 发送到无效地址会发生什么？">
    如果您将 QUAI 发送到无效地址，交易将失败，您的代币应该会被退回。发送前始终仔细检查地址。
  </Accordion>
  
  <Accordion title="我可以将 Quai 钱包 snap 与现有的以太坊 dApp 一起使用吗？">
    Quai 钱包 snap 专为 Quai 交互而设计。对于以太坊 dApp，您将继续使用常规的 MetaMask 钱包。
  </Accordion>
  
  <Accordion title="我在尝试发送 QUAI 时遇到错误。可能是什么问题？">
    常见问题包括 QUAI 余额不足、接收方地址无效或网络连接问题。检查您的余额并验证接收方地址格式。
  </Accordion>
  
  <Accordion title="我可以导出我的 Quai 私钥吗？">
    目前还不行。Quai 钱包 snap 目前不提供对私钥的直接访问。您的 Quai 钱包源自您的 MetaMask 密钥，因此只要您可以访问 MetaMask 账户，就可以访问您的 Quai 钱包。这将在未来版本中更新。
  </Accordion>
</AccordionGroup>
