## 社区频道

通过我们的官方频道与 Quai 网络社区联系：

| 频道 | 用途 | 链接 |
|---------|---------|------|
| Discord | 主要社区中心 | [加入 Discord](https://discord.gg/quai) |
| Twitter | 新闻和更新 | [关注 @QuaiNetwork](https://twitter.com/QuaiNetwork) |
| Telegram | 全球社区聊天 | [加入 Telegram](https://t.me/QuaiNetwork) |
| YouTube | 教育内容 | [订阅](https://youtube.com/@QuaiNetwork) |
| GitHub | 开源代码 | [查看仓库](https://github.com/dominant-strategies) |

## 社区治理

Quai 网络致力于建立一个社区拥有和运营的系统。我们正在开发治理机制，让社区成员在网络的未来发展中拥有发言权。

请关注有关治理提案和投票机制的更新。
