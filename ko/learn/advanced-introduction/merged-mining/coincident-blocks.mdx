## 체인 간의 자연스러운 브리지

주사위를 굴려서 가끔 운이 좋아 여러 게임에서 동시에 이길 수 있다고 상상해보세요. 그것이 바로 일치 블록입니다 - 여러 블록체인 요구사항을 동시에 만족시키는 마이닝 해결책입니다.

**이들을 특별하게 만드는 것:**
- **추가 작업 불필요**: 일반적인 마이닝을 통해 자연스럽게 발생
- **수학적 보장**: 검증자나 위원회에 의존하지 않음
- **완벽한 동기화**: 순수한 확률을 통해 체인들이 연결된 상태 유지

**크로스체인 트랜잭션을 가능하게 하는 방법:**
일치 블록이 발생하면, 체인 간에 깨질 수 없는 수학적 링크를 만듭니다. 이 링크는 무신뢰 브리지 역할을 하여, 중개자 없이 자산과 데이터가 체인 간에 이동할 수 있게 합니다.

## 운의 수학

**간단한 비유:**
- 존 난이도: 6을 굴려야 함
- 리전 난이도: 6을 두 번 굴려야 함
- 프라임 난이도: 6을 세 번 굴려야 함

때로는 하나만 필요할 때 6을 세 번 굴리기도 합니다 - 그것이 일치 블록입니다!

**확률 표:**

| 블록의 "운" 정도 | 발생 확률 |
| -------------------------- | -------------------- |
| 약간 운이 좋음 (0 하나 추가) | 50% |
| 꽤 운이 좋음 (0 둘 추가) | 25% |
| 매우 운이 좋음 (0 셋 추가) | 12.5% |
| 극도로 운이 좋음 (0 넷 추가) | 6.25% |

**이것이 중요한 이유:**
위원회, 검증자, 또는 복잡한 프로토콜에 의존하는 다른 멀티체인 시스템과 달리, Quai의 크로스체인 브리지는 순수한 수학에 의해 만들어집니다. 신뢰가 필요 없습니다 - 단지 확률만 있으면 됩니다.

## 일치 블록 작동 방식

**핵심 원칙:**

1. **계층구조 중요**: 프라임 블록은 항상 일치 (모든 체인을 만족)
2. **독립성 보장**: 존들이 기다리지 않고 블록 생성 가능
3. **원자적 유효성**: 블록이 모든 곳에서 유효하거나 아무 곳에서도 유효하지 않아야 함
4. **완벽한 동기화**: 모든 체인이 일치 블록을 동시에 추가

**체인 계층구조:**
- **프라임 블록**: 항상 일치 (모든 자물쇠를 여는 마스터 키처럼)
- **리전 블록**: 자신의 존들과 일치
- **존 블록**: 일반적으로 독립적, 가끔 일치

**왜 블록 시간이 다른가?**
- 프라임: ~20초 (느리지만 초고보안)
- 리전: ~10초 (균형)
- 존: ~5초 (사용자를 위해 빠름)

이 설계는 존들이 일치 블록을 통한 주기적 동기화를 유지하면서 빠르게 작동할 수 있도록 보장합니다.

## 노드 운영

**최소 요구사항 - "슬라이스":**
마이너가 운영할 수 있는 가장 작은 노드는 슬라이스 노드입니다. 무신뢰로 참여하려면 다음이 필요합니다:
- 프라임 체인 (글로벌 조정)
- 하나의 리전 체인 (지역 조정)
- 하나의 존 체인 (트랜잭션이 발생하는 곳)

이 "슬라이스"는 다른 누구도 신뢰하지 않고 트랜잭션을 검증하는 데 필요한 모든 것을 제공합니다.

**시스템의 아름다움:**
일치 블록은 여러 블록체인이 동기화된 상태를 유지하고 보안을 공유할 수 있는 무신뢰, 자동적, 그리고 수학적으로 보장된 방법을 만듭니다 - 모두 마이닝의 자연스러운 무작위성을 통해서요.
