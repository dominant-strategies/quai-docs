A simple hashing function which operates on UTF-8 strings to compute an 32-byte identifier.

This simply computes the [**UTF-8 bytes**](/ko/sdk/content/functions/toUtf8Bytes) and computes the [**keccak256**](/ko/sdk/content/functions/keccak256).
