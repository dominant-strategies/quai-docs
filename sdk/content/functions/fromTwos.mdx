```ts
function fromTwos(_value, _width): bigint
```

Convert `value` from a twos-compliment representation of `width` bits to its value.

If the highest bit is `1`, the result will be negative.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `_value` | [`BigNumberish`](/sdk/content/type-aliases/BigNumberish) | The value to convert. |
| `_width` | [`Numeric`](/sdk/content/type-aliases/Numeric) | The width of the value in bits. |

## Returns

`bigint`

The value.

## Throws

If the value is too large for the width.
