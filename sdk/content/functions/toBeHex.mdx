```ts
function toBeHex(_value, _width?): string
```

Converts `value` to a Big Endian hexstring, optionally padded to `width` bytes.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `_value` | [`BigNumberish`](/sdk/content/type-aliases/BigNumberish) | The value to convert. |
| `_width`? | [`Numeric`](/sdk/content/type-aliases/Numeric) | The width of the value in bytes. |

## Returns

`string`

The hexstring.

## Throws

If the value exceeds the width.
