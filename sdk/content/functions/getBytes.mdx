```ts
function getBytes(value, name?): Uint8Array
```

Get a typed Uint8Array for `value`. If already a Uint8Array the original `value` is returned; if a copy is required
use [**getBytesCopy**](/sdk/content/functions/getBytesCopy).

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `value` | [`BytesLike`](/sdk/content/type-aliases/BytesLike) | The value to convert to a Uint8Array. |
| `name`? | `string` | The name of the value for error context. |

## Returns

`Uint8Array`

The typed Uint8Array.
