```ts
function toTwos(_value, _width): bigint
```

Convert `value` to a twos-compliment representation of `width` bits.

The result will always be positive.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `_value` | [`BigNumberish`](/sdk/content/type-aliases/BigNumberish) | The value to convert. |
| `_width` | [`Numeric`](/sdk/content/type-aliases/Numeric) | The width of the value in bits. |

## Returns

`bigint`

The value.

## Throws

If the value is too large for the width.
